\doxysubsection{Detailed Description}
\doxylink{class_assignment}{Assignment} owned by a user, linked to a course, with scheduling and workflow status. 

{\ttfamily \doxylink{class_assignment_ad257ca8b777ddec191fd69a3e23adb2f}{save}} maps to {\ttfamily create\+Assignment} on the database (upsert/\+update path in {\ttfamily \doxylink{class_d_b}{DB}}). {\ttfamily course\+\_\+name} is optional UI data from joins, not stored on the assignment row itself.
