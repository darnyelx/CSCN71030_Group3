\doxysubsection{Detailed Description}
Query surface used by tests and future UI features (keyword search, filters, sort). 

Operates on data from {\ttfamily get\+All\+Assignments} then filters/\+sorts in memory. Some filter APIs are stubs and return an empty vector until implemented.
