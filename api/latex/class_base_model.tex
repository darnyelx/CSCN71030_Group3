\doxysubsection{Detailed Description}
Base type for entities mapped to SQL tables ({\ttfamily \doxylink{class_user_model}{User\+Model}}, {\ttfamily \doxylink{class_assignment}{Assignment}}, etc.). 

Concrete models implement {\ttfamily \doxylink{class_base_model_abdc6d5be8ae06dedf6d08fefe1efe2f8}{get\+Table\+Name}} for logging or generic SQL helpers, and {\ttfamily \doxylink{class_base_model_ad64fcc8fdbd3663f8267f12d009683a2}{save}} /\+ {\ttfamily \doxylink{class_base_model_a63f923033f87f2b11f97bff190fdd685}{remove}} to delegate writes to an {\ttfamily \doxylink{class_i_database}{IDatabase}} implementation.
