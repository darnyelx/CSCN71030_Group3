\doxysubsection{Detailed Description}
In-\/memory key/\+value store populated by {\ttfamily \doxylink{class_config_ab9591f046fd7f8186e6b3822afb85529}{load()}} from a path next to the executable. 

Lines are parsed as {\ttfamily KEY=value}; blank lines and {\ttfamily \#} comments are skipped. Keys and values are trimmed of leading/\+trailing whitespace.
