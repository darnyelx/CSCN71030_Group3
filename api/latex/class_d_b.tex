\doxysubsection{Detailed Description}
Process-\/wide database access:\+ one {\ttfamily pqxx::\+connection}, implements all {\ttfamily \doxylink{class_i_database}{IDatabase}} queries. 

The singleton reads {\ttfamily connection\+String} (from environment and/\+or {\ttfamily \doxylink{class_config}{Config}}) before opening the connection. Copying is disabled to preserve a single connection lifecycle.
