\doxysubsection{Detailed Description}
Port for Postgre\+SQL-\/backed operations used by controllers and {\ttfamily \doxylink{class_base_model}{Base\+Model}} implementations. 

Implementations (for example {\ttfamily \doxylink{class_d_b}{DB}}) execute SQL and map rows to the domain types declared in this interface. Controllers depend on this abstraction for testability.
