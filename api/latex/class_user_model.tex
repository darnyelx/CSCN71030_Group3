\doxysubsection{Detailed Description}
Mutable in-\/memory representation of a user row. 

{\ttfamily \doxylink{class_user_model_a067d5e2ae3a2e4eb0c20f2e30a2fcb3e}{save}} delegates to {\ttfamily \doxylink{class_i_database_a27e07395648063e2c8ffef2e5b668f72}{IDatabase::\+create\+User}}. Password field stores a libsodium {\ttfamily crypto\+\_\+pwhash\+\_\+str} output string, not plain text, after registration flows.
